Key Features

1

Natural Language Input

Type "Pay $500 to John, account 9876543210" and all fields fill instantly. No more typing each box.

2

Document Upload

Upload an invoice photo or PDF — AI reads it and extracts all payment details into the form.

3

Conversational Filling

If info is missing, AI asks one question at a time: "What is the routing number?" — like a helpful assistant.

4

Inline AI Help

Click help on any field and ask "What is a routing number?" — AI answers right inside the form.

5

Visual Highlights

All AI-filled fields are highlighted with a colored border — customer can clearly see what was auto-filled.

6

PDF Export

Download the completed form as a PDF for records or offline sharing.

How It Works — Step by Step

1

Customer opens the bank transfer page

A floating AI chat bar appears at the bottom of the existing form — no page redirect needed.

2

Customer describes the transfer

"Send five hundred dollars to John Doe, account 9876543210" — or uploads an invoice image.

3

AI extracts and fills all fields

Name split into First/Last, amount converted from words to digits, currency defaults to USD, fields highlighted.

4

AI asks only for what's missing

If routing number is not provided, AI asks once: "Could you provide the routing number?" Never repeats itself.

5

Customer reviews and submits

The AI never submits the form. The customer always has full control over the final action.

Security & Privacy Rules

  • ✓
    No data stored on server — all form data stays in the customer's browser. When the tab closes, everything is wiped.
  • ✓
    Customer always submits — the AI fills fields but never submits the form. Full control stays with the customer.
  • ✓
    Prompt injection protection — all user input is sanitized before being sent to the AI to prevent malicious commands.
  • ✓
    Session-only memory — the AI does not remember previous conversations. Each session starts fresh.
  • !
    Future: Backend proxy — for enterprise use, API keys should move server-side to prevent exposure in the browser.
